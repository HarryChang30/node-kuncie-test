# NodeJS Kuncie Backend Engineering Test

Take home backend assessment test

## Pre-requisites
1. Node v14.7.0 (LTS)
2. Express
3. GraphQL
4. MySQL

## Running on Local Development
1. `npm install`
2. `npm run start:dev`

## Run Test
1. `npm run test`
